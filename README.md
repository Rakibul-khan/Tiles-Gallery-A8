## Tiles Gallery
A modern and responsive tile showcase platform built with Next.js App Router, where users can explore beautiful tile collections, search tiles, view detailed information, and manage their profiles securely with authentication.

## Project Purpose
Tiles Gallery is designed to provide users with a visually engaging platform for browsing and discovering aesthetic tile designs. The application focuses on modern UI/UX, secure authentication, responsive layouts, and dynamic tile data management using a JSON server.
